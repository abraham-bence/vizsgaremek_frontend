# Frontend Fejlesztői Dokumentáció

## Projekt neve
**Vizsgaremek Frontend**

## Áttekintés
Ez egy **React + TypeScript** alapú webalkalmazás, amely **SCSS**-t használ stíluskezelésre, és a **Vite** buildrendszert alkalmazza fejlesztéshez és fordításhoz.

## Technológiák
- React
- TypeScript
- Vite
- SCSS
- React Router
